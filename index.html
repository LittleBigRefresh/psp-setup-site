Bonsai LBP PSP Setup

Guide:

1. Download Allefresher
2. Download Bonsai (Domain)
3. Download Bonsai (Format)
4. Enable the plugin (Located at /seplugins/Allefresher.prx) from your CFW.
On PRO/ME, this has to be done using some homebrew or a PC.
On ARK-4, use the ARK-4 file manager to enable the plugin.
